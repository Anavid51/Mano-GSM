# 📱 Mano_GSM - Boutique Officielle en Ligne

Bienvenue sur le dépôt officiel de l'application web *Mano_GSM*. Cette plateforme e-commerce moderne et responsive intègre un catalogue dynamique d'iPhones, de systèmes audio haut de gamme (Sonos), ainsi que de vêtements et chaussures de grande marque, le tout calqué sur la charte graphique et l'identité visuelle de la boutique physique.

---

## 🚀 Fonctionnalités Clés

* *🎙️ Commande Vocale Intégrée :* Recherche d'articles mains libres en temps réel grâce à l'API Web Speech Recognition (disponible sur les navigateurs compatibles comme Google Chrome).
* *🛠️ Espace Administrateur Sécurisé :* Un panneau d'administration protégé par mot de passe permettant de modifier instantanément les prix (FCFA) et de basculer le statut des articles ("Disponible" / "En rupture").
* *🎨 Design "Flyer-Accurate" :* Interface utilisateur soignée respectant scrupuleusement la palette de couleurs officielle (Vert, Jaune, Rouge) du flyer de la marque.
* *📱 Responsive & Mobile-First :* Menu burger interactif et grille fluide optimisée pour une navigation tactile rapide sur smartphone.

---

## 🛠️ Technologies Utilisées

* *HTML5* : Structure sémantique de l'application.
* *CSS3* : Variables globales, dégradés linéaires complexes (linear-gradient) et architecture Flexbox/Grid responsive.
* *JavaScript (Vanilla JS)* : Logique métier, manipulation dynamique du DOM, filtrage de base de données locale et intégration de l'API de reconnaissance vocale.
